# Weather SDK for Java

Этот SDK предоставляет простой и удобный способ получения данных о погоде с помощью сервиса OpenWeatherMap.

## Установка

1. **Требования:** Java 11 или выше.
2. **Зависимости:** Проект использует библиотеку Jackson для работы с JSON. Добавьте ее в ваш проект.

   Для Maven:
   ```xml
   <dependency>
       <groupId>com.fasterxml.jackson.core</groupId>
       <artifactId>jackson-databind</artifactId>
       <version>2.15.0</version>
   </dependency>
